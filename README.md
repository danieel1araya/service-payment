Service Payment Java





  A desktop utility bill payment application built in Java with MVC architecture,

  JSON user persistence, and CSV service records.







  
  
  
  




---

## Architecture

| Layer | Responsibility |
|---|---|
| **Model** | User data, services, and file persistence logic |
| **View** | Swing forms — Login, Home, Services, Balance, Profile |
| **Controller** | User input handling and application flow |

---

## Features

- User registration and login with balance tracking
- JSON persistence — user data stored and loaded via Gson
- CSV records — service payment history managed via OpenCSV
- Utility bill payments — electricity, water, internet, TV with balance validation
- Profile management — edit user info and recharge account balance
- Modern UI — FlatLaf dark theme applied globally
